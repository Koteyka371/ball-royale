But `hash` in Python 3 is randomized per-session! This means in tests the result will change depending on the run. Wait, no, `hash` of integers is just the integer itself, but `hash` of strings is randomized!
Better use:
```python
(enemy_id + decoy_id) % 100 > 50
```
This is perfectly deterministic across runs!

```python
# python
```
