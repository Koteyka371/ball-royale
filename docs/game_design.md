# Game Design Document: Ball Royale

## Core Concept

2D арена-батл-рояль с самуправляемыми шариками. **Все шарики управляются встроенным ИИ** — каждое решение (атака, уход, использование скилла, сбор бустера) принимается автономно. Игрок наблюдает за хаосом или минимально вмешивается (ускорение, пауза, фокус на конкретном шарике).

**Ключевой принцип**: Шарик — это живое существо с характером, инстинктами и стратегией. Не робот, executing script, а "живой" боец, который принимает решения на основе ситуации.

## Genre

- Primary: **Auto-battler / Spectator sport**
- Secondary: **Simulation / Chaos engine**
- Reference games:
  - Rusty's Retirement (auto-battler, наблюдают)
  - Legion TD (tower defense с автономными юнитами)
  - Chess engines (ИИ vs ИИ)
  - Battle bots (роботы дерутся сами)

## Target Audience

- Age: 8+
- Platform: **PC + Web**
- Session length: **30 сек — 2 минуты**
- Аудитория YouTube: зрители, которые любят наблюдать за хаосом

## AI System (Main Feature)

### Architecture: Brain per Ball

Каждый шарик имеет собственную "мозговую" систему:

```
┌─────────────────────────────────────────────────────────┐
│                     BALL BRAIN                          │
├─────────────────────────────────────────────────────────┤
│  1. PERCEPTION (Восприятие)                             │
│     - Вижу врагов в радиусе обзора                      │
│     - Вижу союзников                                    │
│     - Вижу бустеры и ловушки                            │
│     - Чувствую своё HP и кулдаун скилла                 │
│                                                          │
│  2. EMOTION (Эмоции)                                    │
│     - Страх (низкое HP → убегаю)                        │
│     - Агрессия (высокое HP → атакую)                    │
│     - Жадность (вижу бустер → иду за ним)               │
│     - Героизм (союзник в опасности → помогаю)           │
│                                                          │
│  3. DECISION (Решение)                                  │
│     - Взвешиваю варианты по приоритету                  │
│     - Учитываю характер и роль                          │
│     - Принимаю решение                                  │
│                                                          │
│  4. ACTION (Действие)                                   │
│     - Движение к цели                                   │
│     - Атака                                            │
│     - Использование скилла                              │
│     - Побег                                             │
└─────────────────────────────────────────────────────────┘
```

### AI Decision Pipeline

```python
# Упрощённый пайплайн решений для каждого шарика
def make_decision(ball, world):
    # 1. Собрать информацию
    threats = find_nearby_enemies(ball, world)
    allies = find_nearby_allies(ball, world)
    boosters = find_nearby_boosters(ball, world)
    
    # 2. Оценить ситуацию
    danger_level = assess_danger(ball, threats)
    opportunity_level = assess_opportunity(ball, boosters, allies)
    
    # 3. Выбрать стратегию на основе HP и характера
    if ball.hp < ball.max_hp * 0.3:
        strategy = "flee"  # Низкое HP → убегаю
    elif danger_level > 0.7:
        strategy = "defend"  # Много врагов → защищаюсь
    elif opportunity_level > 0.5:
        strategy = "opportunistic"  # Есть бустер/слабый враг → атакую
    else:
        strategy = ball.personality  # Зависит от типа шарика
    
    # 4. Выполнить действие
    execute_strategy(ball, strategy, world)
```

### Perception System (Радиус обзора)

Каждый шарик имеет:
- **Radius**: 100-300 пикселей (зависит от типа)
- **FOV**: 360° (видит всё вокруг) или 180° (видит только вперёд)
- **Priority targets**: ближайший враг, самый слабый враг, самый сильный враг

### Emotion System (Эмоциональное состояние)

Каждый шарик имеет эмоциональные модификаторы:

| Эмоция | Условие | Эффект |
|--------|---------|--------|
| **Fear** | HP < 30% | +50% к скорости побега, -30% к атаке |
| **Rage** | HP > 80% и союзник убит | +50% к атаке, -20% к защите |
| **Greed** | Видит бустер | Игнорирует врагов, идёт за бустером |
| **Cowardice** | Первый удар | 30% шанс убежать |
| **Heroism** | Союзник < 30% HP | Идёт на выручку |
| **Bloodlust** | Убил 2+ врагов | Ищет следующую цель |

### Personality Types (Характеры)

Каждый тип шарика имеет уникальный характер:

| Тип | Характер | Поведение |
|-----|----------|-----------|
| **Warrior** | Агрессивный | Ищет бой, атакует первым, не убегает |
| **Scout** | Любопытный | Исследует, атакует слабых, убегает от сильных |
| **Tank** | Храбрый | Защищает союзников, принимает урон |
| **Healer** | Заботливый | Следует за союзниками, лечит раненых |
| **Sniper** | Осторожный | Держится на расстоянии, атакует издалека |
| **Bomber** | Безрассудный | Подрывается на врагов, жертвует собой |
| **Ninja** | Хитрый | Скрывается, атакует сзади, убегает |
| **King** | Лидер | Усиливает союзников, держится позади |

### Decision Examples

**Пример 1: Warrior с 50% HP**
```
Вижу: 3 врага, 1 союзник, 0 бустеров
Эмоция: neutral (HP > 30%)
Характер: агрессивный
Решение: атакую ближайшего врага
```

**Пример 2: Healer с 20% HP**
```
Вижу: 2 врага, 1 союзник на 40% HP
Эмоция: fear (HP < 30%)
Характер: заботливый
Решение: убегаю + лечу союзника (если возможно)
```

**Пример 3: Sniper на позиции**
```
Вижу: 1 враг на средней дистанции, 2 союзника впереди
Эмоция: calm
Характер: осторожный
Решение: атакую издалека, не приближаюсь
```

**Пример 4: Bomber vs толпа**
```
Вижу: 5 врагов рядом
Эмоция: bloodlust (убил 2+ врагов)
Характер: безрассудный
Решение: подрываюсь на всех (suicide bomb)
```

## AI Behaviors (Поведения)

### Combat Behaviors (Боевые)

1. **Aggressive Chase** — преследует ближайшего врага
2. **Target Weak** — ищет самого слабого врага
3. **Target Strong** — атакует самого сильного (для Tank)
4. **Kite** — держит дистанцию, атакует при приближении (для Sniper)
5. **Flank** — обходит сзади для удара (для Ninja)
6. **Group Attack** — атакует вместе с союзниками
7. **Ambush** — прячется, ждёт подходящий момент

### Defensive Behaviors (Защитные)

1. **Flee** — убегает от врагов
2. **Circle Strafe** — кружит вокруг врага, уклоняясь
3. **Use Shield** — активирует щит при ударе
4. **Retreat to Ally** — отступает к союзнику- Tank
5. **Hide Behind** — прячется за другими шариками

### Support Behaviors (Поддержка)

1. **Heal Ally** — лечит ближайшего раненого союзника
2. **Buff Ally** — усиливает союзника перед боем
3. **Body Block** — встаёт между союзником и врагом
4. **Escort** — сопровождает раненого союзника

### Utility Behaviors (Утилитарные)

1. **Collect Booster** — идёт за бустером
2. **Avoid Trap** — обходит ловушки
3. **Reposition** — перемещается на лучшую позицию
4. **Wait and Watch** — ждёт, пока другие подраются

## AI Difficulty Levels

### Easy (Для новичков)
- Медленные реакции (0.5 сек задержка)
- Простые решения (только 2-3 варианта)
- Не используют скиллы оптимально
- Часто ошибаются

### Medium (Стандарт)
- Нормальные реакции (0.2 сек задержка)
- Средние решения (4-5 вариантов)
- Используют скиллы когда нужно
- Иногда ошибаются

### Hard (Для профессионалов)
- Быстрые реакции (0.05 сек задержка)
- Оптимальные решения (все варианты)
- Используют скиллы идеально
- Почти не ошибаются

### Chaos (Максимальный хаос)
- Невозможные реакции (мгновенно)
- Абсурдные решения (случайные)
- Спамят скиллами
- Создают максимально забавный хаос

## Ball Types (Expanded)

### Warrior (Агрессор)
- **HP**: 150
- **ATK**: 20
- **SPD**: 4
- **Skill**: Wave Attack (AoE вперёд)
- **AI**: Агрессивный, ищет ближайшего врага, не убегает
- **Приоритет**: Атака > Защита > Бустеры

### Scout (Разведчик)
- **HP**: 80
- **ATK**: 10
- **SPD**: 8
- **Skill**: Dash (быстрый рывок)
- **AI**: Любопытный, исследует, атакует слабых
- **Приоритет**: Бустеры > Слабые враги > Сильные враги

### Tank (Танк)
- **HP**: 250
- **ATK**: 8
- **SPD**: 3
- **Skill**: Shield (блокирует урон)
- **AI**: Храбрый, защищает союзников
- **Приоритет**: Защита союзников > Атака > Бустеры

### Healer (Целитель)
- **HP**: 100
- **ATK**: 5
- **SPD**: 5
- **Skill**: Heal (восстанавливает HP)
- **AI**: Заботливый, следует за ранеными
- **Приоритет**: Лечение > Защита > Атака

### Sniper (Снайпер)
- **HP**: 70
- **ATK**: 30
- **SPD**: 3
- **Skill**: Snipe (точный выстрел издалека)
- **AI**: Осторожный, держит дистанцию
- **Приоритет**: Дальняя атака > Уход > Близкий бой

### Bomber (Подрывник)
- **HP**: 60
- **ATK**: 50 (один раз)
- **SPD**: 4
- **Skill**: Bomb ( AoE взрыв, жертвует собой)
- **AI**: Безрассудный, ищет толпу врагов
- **Приоритет**: Максимум врагов в AoE > Смерть с эффектом

### Ninja (Ниндзя)
- **HP**: 90
- **ATK**: 25
- **SPD**: 7
- **Skill**: Stealth (невидимость 3 сек)
- **AI**: Хитрый, атакует сзади, убегает
- **Приоритет**: Атака сзади > Уход > Прямой бой

### King (Король)
- **HP**: 120
- **ATK**: 15
- **SPD**: 4
- **Skill**: Command (+20% ATK/SPD союзникам)
- **AI**: Лидер, держится позади, усиливает команду
- **Приоритет**: Поддержка > Защита > Атака

## Interaction Modes (Взаимодействие игрока)

### Spectator Mode (Наблюдатель)
- Игрок наблюдает за всем боем
- Может переключать камеру между шариками
- Может ускорять/замедлять время
- Может добавлять "события" (новые бустеры, изменения арены)

### Limited Control (Ограниченное управление)
- Игрок может кликнуть на шарика
- Выбрать для него "режим" (агрессия/защита/поддержка)
- ИИ продолжает принимать решения, но с учётом приказа

### Coach Mode (Тренер)
- Игрок задаёт общую стратегию команды
- ИИ шариков следует стратегии
- Примеры: "Атакуйте!", "Защищайтесь!", "Собирайте бустеры!"

## YouTube/Shorts Strategy

### Video Format
- **Duration**: 30-60 seconds
- **Format**: Vertical (9:16)
- **Camera**: Auto-follows most exciting action
- **UI**: Minimal, shows only health bars and kill feed

### Viral Moments
1. **Clutch plays** — шарик выживает с 1 HP
2. **Comebacks** — команда проигрывает, но побеждает
3. **Funny fails** — шарик падает в ловушку
4. **Epic kills** — идеальный скилл
5. **1v1 Finals** — финальный бой двух шариков
6. **Team wipes** — одна команда уничтожает другую

### Camera System
- **Auto-follow**: Камера следует за самым активным шариком
- **Highlight detection**: Автоматически фиксирует эпичные моменты
- **Replay**: Повтор лучших моментов в замедленном воспроизведении
